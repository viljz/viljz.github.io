    • Sublime常用插件安装

    1. 打开 Sublime Text3，依次选择：首选项 Package Control

    2. 在弹出的 Package Control 管理面板中输入 Inatall Package 进入插件安装界面

    3. 在弹出的 Package Control 管理面板中输入 Remove Package 进入插件卸载界面

    1. ChineseLocalizations
      • 功能：sublime汉化插件
      • 简介：多国语言切换
    1. Emmet
      • 功能：编码快捷键，前端必备
      • 简介：Emmet作为zen coding的升级版，对于前端来说，可是必备插件，如果你对它还不太熟悉，可以在其官网上看下具体的演示视频。
    1. JSFormat
      • 功能：JavaScript的代码格式化插件
      • 简介：很多网站的js代码都进行了压缩，一行式的甚至混淆压缩，这让我们看起来很吃力。而这个插件能帮我们把原始代码进行格式的整理，包括换行和缩进等等，让代码一目了然，更快读懂
      • 使用：在已压缩的JS文件中，右键选择jsFormat或者使用默认快捷键 Ctrl + Alt + F
    1. LESS
      • 功能：LESS高亮插件
      • 简介：用LESS的同学都知道，sublime没有支持less的语法高亮，所以这个插件可以帮上我们
      • 使用：打开.less文件或者设置为less格式
    1. jQuery
      • 功能：jQ函数提示
      • 简介：快捷输入jQ函数，是偷懒的好方法
    1. DocBlockr
      • 功能：生成优美注释
      • 简介：标准的注释，包括函数名、参数、返回值等，并以多行显示，手动写比较麻烦
      • 使用：输入/**然后回车
    1. Nodejs
      • 功能：node代码提示
      • 教程：Nodejs
    1. Terminal
      • 功能： 在编辑的文件目录打开cmd，默认快捷键 ctrl + shift + t
      • 配置： Preferences => Package setting => Terminal => Setting-User
    1. SideBarEnhancements
      • 功能: 增加文件右击的操作功能
      • 使用: FOLDRES 鼠标右键
