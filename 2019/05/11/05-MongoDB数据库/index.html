    • MongoDB 数据库概念,配置已经使用

    关系型数据库

    • 所有的关系型数据库都需要通过sql 语言来操作
    • 所有的关系型数据库在操作之前都需要设计表结构
    • 而且数据表还支持约束
      • 唯一的
      • 主键
      • 默认值
      • 非空

    非关系型数据库

    • 非关系型数据库非常的灵活
    • 有的非关系型数据库就是 key 键值对
    • MongoDB 是长的最像关系型数据库的非关系型数据
      • 数据库 -> 数据库
      • 数据表 -> 集合(数据)
      • 表记录 -> (文档对象)
    • MongoDB 不需要设计表结构
    • 可以任意往里面存数据, 没有结构性这一说法
